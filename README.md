📌 Overview

This folder contains my Data Structures & Algorithms (DSA) practice in JavaScript.
It’s structured by topics & patterns, with solutions to LeetCode, HackerRank, and other coding challenges.


Why I’m doing this:


   💡 Build strong problem-solving skills and coding intuition

   🏆 Prepare for coding interviews at top tech companies

   📚 Understand core concepts in arrays, strings, recursion, DP, trees, graphs, etc.

   🔄 Practice multiple approaches for the same problem to improve efficiency

   📈 Track progress across topics and patterns
   

⚡ How I Approach Problems

Read problem carefully → identify pattern (e.g., sliding window, recursion, DP).

Write brute-force solution first, then optimize.

Track time and space complexity.

Add edge cases & test cases in comments.

Learn multiple approaches if applicable.
